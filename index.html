1.2. Object prototype 


									let x = {};

									let x = new Object().

if  Object  has a prototype property, then new operator uses it to set [[Prototype]] for the new object x.

1.2. Object prototype 


								Object.getPrototypeOf(x);

								x.__proto__;
								//{constructor: ƒ, __defineGetter__: ƒ, __defineSetter__: ƒ, …}

								x.toString(); 
								// [object Object]

								This prototype chain is only one link long.
								x -> Object. 

								x.__proto__.__proto__; 
								// Null

1.3. Array prototype


									let y = [];

									Turns out that’s just sugar over creating 
									a new instance of the Array class.

									let y = new Array().

									y.__proto__;
									// [constructor: ƒ, concat: ƒ, pop: ƒ, push: ƒ, …]

									y-> Array -> Object.

									y.__proto__.__proto__;
									// {constructor: ƒ, __defineGetter__: ƒ, __defineSetter__: ƒ, …}

1.3. Array prototype


									y.__proto__ === Array.prototype;
									// true
									
									y.__proto__.__proto__ === Object.prototype;
									// true

									Array.prototype.isPrototypeOf(y);
									// true
									
									Object.prototype.isPrototypeOf(Array);
									// true

									y instanceof Array;
									// true

2.3. Iteration methods 


									Array.prototype.forEach()

									//a[1] = 2
									//a[2] = 4
									//a[3] = 5
									//a[4] = 6
									//a[5] = 7

2.3. Iteration methods
